4.	В поле поиск ввести значение ПНЕВМОШЛИФМАШИНА  и получить результаты поиска.
5.	Кликнуть на кнопку Показать контакты для отображения контактов.
6.	Перейти в блок Новости, обновить страницу и вернуться на главную страницу.

9.	Проверка нахождения элемента Акции.
10.	Проверка действия кнопки купить для не авторизованного пользователя.
